Soft Mobility
=============

SoftMobility computes the mobility and dynamics of deformable assemblies of
spheres in Stokes flows. It is designed for scientific workflows where the same
model should be simulated, differentiated, and optimized with JAX.

.. tip::

   You can try the tutorials directly in your browser via Google Colab — no
   local clone or install required. See
   :ref:`Running tutorials on Google Colab <running-tutorials-on-google-colab>`
   for the full list of badges and instructions.
