============
Contributing
============

Contributions are welcome and appreciated.

The complete developer guide — environment setup, running the tests,
building the documentation, the versioning scheme, the release process, and
the policy for opening issues and pull requests — lives in the Sphinx
documentation under ``docs/source/developers.rst`` (rendered as the
*Developers* page).

Quick links:

- Issue tracker: https://github.com/celoy/SoftMobility/issues
- Source repository: https://github.com/celoy/SoftMobility
